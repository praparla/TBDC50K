## Active Backlog

### P1 - Route & Navigation
- [ ] **Elevation profile** - Show elevation chart along the route with grade indicators
- [ ] **Turn-by-turn segments** - Break route into stop-to-stop segments with individual distances
- [ ] **Alternative route suggestions** - Allow toggling between different route options between stops
- [ ] **Offline map support** - Cache tiles for race-day use without cell service

### P1 - Race Planning
- [ ] **Pace calculator** - Input target finish time, get per-segment pace targets
- [ ] **Aid station / restroom finder** - Auto-populate public restrooms and water fountains near the route
- [ ] **Time-of-day estimator** - Show estimated arrival time at each stop based on pace
- [ ] **Weather overlay** - Pull weather forecast for race day

### P2 - Custom Pins & Social
- [ ] **Pin categories & filtering** - Toggle visibility of pin types (restrooms, friends, food, etc.)
- [ ] **Shareable pin sets** - Export/import pin collections to share with running partners
- [ ] **Pin notes & photos** - Add notes or photos to custom pins
- [ ] **Friend location sharing** - Real-time location sharing on race day

### P2 - Data & Tracking
- [ ] **Strava integration** - Import training runs, overlay on race route
- [ ] **GPX export** - Export the route with custom pins as a GPX file
- [ ] **Race-day tracking mode** - GPS tracking with live position on the map
- [ ] **Split history** - Log actual splits at each stop during the race

### P2 - UI/UX
- [ ] **Mobile-first redesign** - Better mobile experience with collapsible sidebar
- [ ] **Search for locations** - Geocoding to search and add pins by address
- [ ] **Drag to reposition pins** - Allow dragging placed pins to adjust location
- [ ] **Dark/light mode toggle** - Theme switching

### P3 - Extras
- [ ] **Taco Bell menu integration** - Show menus at each stop, plan what to eat
- [ ] **Calorie tracker** - Track calories consumed vs burned during the race
- [ ] **Photo gallery** - Race-day photo upload tied to map locations
- [ ] **Leaderboard** - Compare times with other participants
